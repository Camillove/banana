02

Sintaxe Básica

A estrutura fundamental de todo programa Java.

📄

Arquivo .java

Todo código Java fica em arquivos .java. O nome do arquivo deve ser igual ao nome da classe pública.

🏛️

Classe

Todo código Java precisa estar dentro de uma classe. É a unidade básica de organização.

▶️

Método main

O ponto de entrada de qualquer programa Java. É onde a execução começa.

🖨️

System.out.println

Exibe texto no console. O mais usado para testes e debugging no início.

OlaMundo.java 
// Meu primeiro programa Java
public class OlaMundo {

    public static void main(String[] args) {
        System.out.println("Olá, Mundo!");
        System.out.println("Java é incrível!");
    }

}
💡

Dica: Java é case-sensitive — Main e main são coisas diferentes! Todo statement termina com ;

04

Operadores

Operadores aritméticos, relacionais e lógicos.

➕

Aritméticos

+ adição   - subtração
* multiplicação   / divisão
% módulo (resto)

⚖️

Relacionais

== igual   != diferente
> maior   < menor
>= maior/igual   <= menor/igual

🔗

Lógicos

&& E lógico (AND)
|| OU lógico (OR)
! NÃO lógico (NOT)

Operadores.java 
int a = 10, b = 3;

System.out.println(a + b);   // 13
System.out.println(a - b);   // 7
System.out.println(a * b);   // 30
System.out.println(a / b);   // 3  (divisão inteira)
System.out.println(a % b);   // 1  (resto da divisão)

// Lógicos
boolean x = 5 > 3 && 10 < 20;  // true
boolean y = 5 > 10 || 1 == 1;  // true
05

Controle de Fluxo

Tomando decisões com if, else e switch.

Condicionais.java — if / else if / else 
int nota = 75;

if (nota >= 90) {
    System.out.println("Conceito A");
} else if (nota >= 70) {
    System.out.println("Conceito B");
} else if (nota >= 50) {
    System.out.println("Conceito C");
} else {
    System.out.println("Reprovado");
}
Switch.java 
int dia = 3;

switch (dia) {
    case 1: System.out.println("Segunda"); break;
    case 2: System.out.println("Terça");   break;
    case 3: System.out.println("Quarta");  break;
    case 4: System.out.println("Quinta");  break;
    case 5: System.out.println("Sexta");   break;
    default: System.out.println("Fim de semana");
}
⚠️

Atenção: Nunca esqueça o break no switch, senão ele continua executando os próximos cases (comportamento "fall-through").

06

Laços de Repetição

Repetindo ações com for, while e do-while.

Loops.java — for 
// for: quando você sabe quantas vezes repetir
for (int i = 1; i <= 5; i++) {
    System.out.println("Contador: " + i);
}
// Saída: 1, 2, 3, 4, 5

// Tabuada do 7
for (int i = 1; i <= 10; i++) {
    System.out.println("7 x " + i + " = " + (7 * i));
}
Loops.java — while & do-while 
// while: quando você não sabe quantas repetições
int x = 1;
while (x <= 5) {
    System.out.println("x = " + x);
    x++;
}

// do-while: executa ao menos 1 vez
int y = 10;
do {
    System.out.println("y = " + y);
    y++;
} while (y < 5);
// Executa 1x mesmo que y >= 5
07

Métodos

Criando blocos de código reutilizáveis.

🔧

Sem retorno (void)

Executa ações mas não devolve valor. Ideal para exibir mensagens e processar dados.

🔄

Com retorno

Processa dados e devolve um resultado usando a palavra return.

📥

Com parâmetros

Recebe valores externos para trabalhar — deixa o método genérico e reutilizável.

Metodos.java 
public class Metodos {

    // Método sem retorno (void)
    static void saudacao(String nome) {
        System.out.println("Olá, " + nome + "!");
    }

    // Método com retorno
    static int somar(int a, int b) {
        return a + b;
    }

    // Método com lógica
    static boolean ehPar(int n) {
        return n % 2 == 0;
    }

    public static void main(String[] args) {
        saudacao("Ana");             // Olá, Ana!
        int resultado = somar(8, 5); // 13
        System.out.println(ehPar(4)); // true
    }
}
08

Arrays

Armazenando múltiplos valores em uma única variável.

📌

Arrays em Java têm tamanho fixo e os índices começam em 0. O primeiro elemento é array[0], o segundo é array[1], e assim por diante.

Arrays.java 
// Declarando e inicializando
int[] numeros = {10, 20, 30, 40, 50};
String[] frutas = {"Maçã", "Banana", "Laranja"};

// Acessando elementos
System.out.println(numeros[0]);   // 10
System.out.println(frutas[1]);    // Banana
System.out.println(numeros.length); // 5 (tamanho)

// Percorrendo com for
for (int i = 0; i < numeros.length; i++) {
    System.out.println(numeros[i]);
}

// For-each (mais limpo!)
for (String fruta : frutas) {
    System.out.println(fruta);
}

Orientação a Objetos

O coração do Java: Classes, Objetos e os 4 pilares da POO.

🏗️

Encapsulamento

Protege os dados da classe usando private e expõe via métodos getters/setters.

🧬

Herança

Uma classe filha herda atributos e métodos da classe pai com extends.

🎭

Polimorfismo

O mesmo método se comporta diferente dependendo do objeto que o chama.

🔷

Abstração

Esconde detalhes de implementação e mostra apenas o essencial.

Pessoa.java — Classe com encapsulamento 
public class Pessoa {

    // Atributos privados
    private String nome;
    private int idade;

    // Construtor
    public Pessoa(String nome, int idade) {
        this.nome = nome;
        this.idade = idade;
    }

    // Getters
    public String getNome() { return nome; }
    public int getIdade() { return idade; }

    // Método
    public void apresentar() {
        System.out.println("Olá! Sou " + nome
            + " e tenho " + idade + " anos.");
    }
}

// Usando a classe:
Pessoa p = new Pessoa("Carlos", 30);
p.apresentar();  // Olá! Sou Carlos e tenho 30 anos.
Herança.java 
// Classe pai
public class Animal {
    public void som() {
        System.out.println("...");
    }
}

// Classe filha herda de Animal
public class Cachorro extends Animal {
    @Override
    public void som() {
        System.out.println("Au au!");
    }
}

public class Gato extends Animal {
    @Override
    public void som() {
        System.out.println("Miau!");
    }
}
