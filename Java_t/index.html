📦 Tipos Primitivos

Java tem 8 tipos primitivos que armazenam valores simples diretamente na memória:

  • int — números inteiros (-2 bilhões a 2 bilhões)
  • double — números decimais (ponto flutuante)
  • boolean — verdadeiro ou falso
  • char — um único caractere
  • long — inteiros muito grandes
  • float — decimal com menos precisão
  • byte / short — inteiros pequenos
JAVA 
// Declaração de variáveis em Java
int idade = 25;
double salario = 3500.50;
boolean ativo = true;
char letra = 'A';
String nome = "Maria"; // Não é primitivo, é objeto

// Imprimir no console
System.out.println("Nome: " + nome);
System.out.println("Idade: " + idade);

➕ Operadores

Operadores permitem realizar cálculos e comparações:

JAVA 
int a = 10, b = 3;

// Aritméticos
System.out.println(a + b);  // 13
System.out.println(a - b);  // 7
System.out.println(a * b);  // 30
System.out.println(a / b);  // 3 (inteiro!)
System.out.println(a % b);  // 1 (resto)

// Relacionais — retornam boolean
System.out.println(a > b);   // true
System.out.println(a == b);  // false
💡 Dica: Divisão entre dois int sempre resulta em inteiro. Para obter decimal, use double a = 10.0 / 3.

🔀 if / else

Executa blocos de código com base em condições:

JAVA 
int nota = 75;

if (nota >= 90) {
    System.out.println("Excelente!");
} else if (nota >= 70) {
    System.out.println("Aprovado!");
} else {
    System.out.println("Reprovado.");
}
// Saída: "Aprovado!"

🎛️ switch

Alternativa ao if/else para múltiplas opções com um mesmo valor:

JAVA 
int dia = 3;

switch (dia) {
    case 1:
        System.out.println("Segunda-feira");
        break;
    case 2:
        System.out.println("Terça-feira");
        break;
    case 3:
        System.out.println("Quarta-feira");
        break;
    default:
        System.out.println("Outro dia");
}
// Saída: "Quarta-feira"

🔁 for

Quando você sabe quantas vezes repetir:

JAVA 
// Conta de 1 a 5
for (int i = 1; i <= 5; i++) {
    System.out.println("Número: " + i);
}

// Percorrendo um array
int[] numeros = {10, 20, 30, 40};
for (int n : numeros) {
    System.out.println(n);
}

🔄 while e do-while

Quando a quantidade de repetições é indefinida:

JAVA 
// while — verifica ANTES de executar
int cont = 0;
while (cont < 3) {
    System.out.println("Contagem: " + cont);
    cont++;
}

// do-while — executa PELO MENOS uma vez
int x = 10;
do {
    System.out.println("x = " + x);
    x++;
} while (x < 5); // Imprime uma vez mesmo assim
💡 Dica: Use break para sair do laço e continue para pular para a próxima iteração.

🧩 Declarando Métodos

Métodos organizam o código em blocos reutilizáveis:

JAVA 
public class Calculadora {

    // Método que retorna a soma
    public static int somar(int a, int b) {
        return a + b;
    }

    // Método que não retorna nada (void)
    public static void saudar(String nome) {
        System.out.println("Olá, " + nome + "!");
    }

    public static void main(String[] args) {
        int resultado = somar(5, 3);
        System.out.println(resultado); // 8
        saudar("Dev");            // Olá, Dev!
    }
}

🏛️ Classes e Objetos

Uma classe é um modelo; um objeto é uma instância desse modelo:

JAVA 
// Definindo a classe
public class Pessoa {
    // Atributos
    String nome;
    int idade;

    // Construtor
    public Pessoa(String nome, int idade) {
        this.nome = nome;
        this.idade = idade;
    }

    // Método
    public void apresentar() {
        System.out.println(
            "Oi! Sou " + nome + ", tenho " + idade + " anos."
        );
    }
}

// Usando a classe
Pessoa p1 = new Pessoa("Ana", 30);
p1.apresentar(); // Oi! Sou Ana, tenho 30 anos.
💡 OOP: Os 4 pilares da Orientação a Objetos em Java são Encapsulamento, Herança, Polimorfismo e Abstração.

☕ JVM, JDK e JRE

JDK (Java Development Kit) — kit para desenvolver em Java.
JRE (Java Runtime Environment) — para executar programas.
JVM (Java Virtual Machine) — executa o bytecode Java em qualquer SO.

TERMINAL 
# Verificar instalação
java -version
javac -version

# Compilar
javac Main.java

# Executar
java Main

🔒 Encapsulamento

Protege os dados de uma classe usando private. O acesso é feito por getters e setters.

JAVA
private String nome;

public String getNome() {
    return nome;
}

public void setNome(
    String nome
) {
    this.nome = nome;
}

🧬 Herança

Uma classe pode herdar atributos e métodos de outra usando a palavra-chave extends.

JAVA
class Animal {
    void comer() {
        System.out
          .println("Comendo");
    }
}

class Cachorro
    extends Animal {
    void latir() {
        System.out
          .println("Au au!");
    }
}

🛡️ Exceções

Tratamento de erros em tempo de execução com try/catch/finally para evitar que o programa quebre.

JAVA
try {
    int r = 10 / 0;
} catch (
    ArithmeticException e
) {
    System.out.println(
        "Erro: " + e.getMessage()
    );
} finally {
    System.out.println(
        "Sempre executa"
    );
}
